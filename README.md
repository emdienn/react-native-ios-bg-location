# react-native-ios-bg-location

React-Native iOS Background Location Plugin

## Installation

```sh
npm install react-native-ios-bg-location
```

## Usage

```js
import { multiply } from "react-native-ios-bg-location";

// ...

const result = await multiply(3, 7);
```
